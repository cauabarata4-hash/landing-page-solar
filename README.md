# 🌞 SolarEnergy - Landing Page

Uma landing page responsiva e moderna desenvolvida para uma empresa fictícia do setor de energia solar. O projeto foi criado para praticar estruturação de páginas modernas, design responsivo e organização de interface voltada para negócios.

## 📋 Características

✅ **Design Responsivo** - Funciona perfeitamente em dispositivos móveis, tablets e desktops
✅ **Menu Hamburger** - Navegação inteligente que se adapta a telas menores
✅ **Animações Suaves** - Transições elegantes ao fazer scroll
✅ **Formulário Validado** - Captura de leads com validação completa
✅ **Formatação Automática** - Máscaras de input para telefone
✅ **Mensagens de Sucesso** - Feedback visual ao usuário
✅ **SEO Otimizado** - Meta tags e estrutura semântica
✅ **Paleta de Cores Moderna** - Laranja e dourado em harmonia com o tema solar

## 🎨 Seções da Landing Page

### 1. **Navegação Sticky**
- Logo com ícone de sol animado
- Menu responsivo com links suavizados
- Hamburger menu para mobile
- Links ativos destacados durante scroll

### 2. **Hero Section**
- Título chamativo com subtítulo
- Ilustração animada (círculo gradient)
- Call-to-action button
- Layout responsivo flexível

### 3. **Serviços**
- 4 cards de serviços principais
- Ícones e descrições
- Efeito hover com elevação
- Grid automático responsivo

### 4. **Benefícios**
- Lista com 6 benefícios-chave
- Checkmarks estilizados com gradient
- Animação ao scroll
- Layout em grid adaptável

### 5. **Depoimentos**
- 3 cards com avaliações 5 estrelas
- Bordas coloridas
- Feedback visual ao hover
- Nomes dos clientes destacados

### 6. **Formulário de Contato**
- Campos: Nome, Email, Telefone, Cidade, Tipo, Mensagem
- Validação completa em tempo real
- Formatação automática de telefone
- Contador de caracteres
- Mensagem de sucesso animada

### 7. **Footer**
- Informações da empresa
- Links rápidos
- Contato e localização
- Design escuro elegante

## 🚀 Como Usar

1. **Clone o repositório**
```bash
git clone https://github.com/cauabarata4-hash/landing-page-solar.git
cd landing-page-solar
```

2. **Abra no navegador**
   - Abra o arquivo `index.html` em um navegador moderno

3. **Customize**
   - Edite o `index.html` para alterar conteúdo
   - Modifique `styles.css` para mudar cores e estilos
   - Atualize `script.js` para adicionar novas funcionalidades

## 🎯 Funcionalidades JavaScript

### Menu Responsivo
```javascript
- Toggle do hamburger menu
- Fechamento automático ao clicar em um link
- Suporte touch para mobile
```

### Validação de Formulário
```javascript
- Validação de campos obrigatórios
- Regex para email e telefone
- Mensagem de erro intuitiva
- Feedback de sucesso animado
```

### Formatação Automática
```javascript
- Máscara de telefone: (XX) 9XXXX-XXXX
- Entrada em tempo real
- Remove caracteres inválidos
```

### Animações ao Scroll
```javascript
- Intersection Observer para eficiência
- Animações de slide-up nos cards
- Navegação ativa durante scroll
- Parallax suave na hero
```

### Outros Recursos
```javascript
- Scroll suave para âncoras
- Contador de caracteres na mensagem
- Detecção de dispositivo móvel
- Links de navegação inteligentes
```
